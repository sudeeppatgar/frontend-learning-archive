# Frontend Learning Archive

This repository is a collection of frontend learning projects built while practicing HTML, CSS, JavaScript, and React.

## Goal

- Practice UI building from scratch.
- Learn component-based development with React + Vite.
- Experiment with layouts, forms, auth-style UI, dashboards, and mini apps.
- Keep all practice work in one archive for revision.

## Repository Structure

```text
frontend-learning-archive/
├── html-css-basics/
├── javascript-experiments/
└── react-practice/
```

## 1) HTML/CSS Basics

Path: `html-css-basics/`

Projects:
- `Amazon-Clone` - Amazon style homepage UI practice.
- `Cardekho-homepage` - CarDekho inspired homepage layout.
- `bootstrap` - Bootstrap-based layout/components experiments.
- `sale-management` - Static dashboard-style sales layout.
- `Login-Forms/login-form-1`
- `Login-Forms/login-form-2`
- `Login-Forms/login-form-3`
- `flex-practice/flex-1`
- `flex-practice/flex-2`
- `flex-practice/flex-3`

What this section teaches:
- CSS fundamentals (box model, spacing, typography).
- Flexbox alignment and responsive patterns.
- Page and component cloning for real-world UI practice.

How to run:
- Open each `index.html` in browser.
- Note: `sale-management` uses `inde.html` as entry file.

## 2) JavaScript Experiments

Path: `javascript-experiments/`

Projects:
- `Reg-Login-Page` - Registration/login UI behavior practice.
- `rok-pap-sis-game` - Rock-paper-scissors game logic in JavaScript.

What this section teaches:
- DOM manipulation and event handling.
- Conditional logic and state updates.
- Wiring CSS/HTML with JS behavior.

How to run:
- Open each project's `index.html` in browser.

## 3) React Practice Projects

Path: `react-practice/`

Projects:
- `Dashboard` - Dashboard UI and component composition.
- `E-commerce` - Product/cart style React app.
- `Sales-management` - Sales/campaign style React screens.
- `CarDekho-Clone` - React clone project (includes Storybook scripts).
- `Food-page` - Food listing/page UI with React.
- `Mbl-application` - Mobile app style UI in React (includes Storybook scripts).
- `React-practice` - General React + TypeScript practice playground.
- `School-info/detail-form/details` - School detail form app in React + TypeScript.

What this section teaches:
- React components, props, and state.
- Project structuring with Vite.
- Routing/UI composition patterns.
- TypeScript basics in React apps.

How to run any React app:

```bash
cd react-practice/<project-folder>
npm install
npm run dev
```

Then open the local URL shown in terminal (usually `http://localhost:5173`).

## Suggested Learning Path

1. Start with `html-css-basics` to build layout confidence.
2. Move to `javascript-experiments` for interactivity and logic.
3. Continue with `react-practice` for component architecture and modern frontend workflow.

## Notes

- This is a learning archive, so coding style and structure may vary between projects.
- Some projects are UI-focused prototypes and may not include backend integration.
- Asset-heavy folders are included intentionally for practice.
